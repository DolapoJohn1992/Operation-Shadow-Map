# Operation Shadow Map: Perimeter Assessment & Vulnerability Audit

## Project Overview

This project serves as a demonstration of active reconnaissance, network discovery, and vulnerability auditing within a simulated corporate DMZ. The objective was to identify exposed assets, map software versions, and prioritize misconfigurations based on real-world risk.

## Technical Stack

* **Operating System:** Ubuntu Linux
* **Tools:** Nmap, Nikto, Bash
* **Environment:** VirtualBox, VS Code, Git/GitHub

## Project Phases

1. **Subnet Sweeping (Active Recon):** Identified live assets on a target /24 subnet using Nmap ping sweeps and aggressive service version scanning.
2. **Vulnerability Auditing:** Conducted targeted web server scanning using Nikto to discover outdated software versions and missing security headers.
3. **Risk Triage:** Prioritized identified vulnerabilities based on likelihood of exploitation and potential impact, producing a formal risk justification.

## Portfolio Artifacts

The repository includes the professional Perimeter Assessment Report detailing the technical findings, service breakdowns, and prioritized executive summary.
